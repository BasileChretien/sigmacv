# Vendored CSL assets

This directory holds CSL citation **styles** (`styles/*.csl`) and **locales**
(`locales/locales-*.xml`), downloaded by `npm run fetch-csl`.

These files come from the Citation Style Language project and are licensed
**CC BY-SA 3.0** (see the repository `NOTICE`). They are data, not code — the
Apache-2.0 license of SigmaCV's source is unaffected.

Do not edit by hand; re-run `npm run fetch-csl` to refresh or add styles
(edit the `STYLES` list in `scripts/fetch-csl.mjs`).
